One Training Cycle — 6 Steps

01
Observe
Read vehicle counts on the 4 incoming lanes of intersection B1. This is the state (shape 4,).
02
Decide
Agent picks action 0 (N-S green) or 1 (E-W green). Early on: random. Later: uses neural net.
03
Act
SumoEnv sends the chosen phase string to SUMO via TraCI (e.g. "GGggrrrr…").
04
Simulate
SUMO runs 5 seconds of traffic with that phase. Vehicles move, some stop and wait.
05
Reward
Reward = −(total waiting time). Less waiting = higher reward. Agent wants to maximize this.
06
Learn
Experience stored in buffer. Agent samples 32 past experiences, runs backprop, updates weights.

🤖 DQNAgent (dqn_numpy.py)
act(state)
Picks an action. Uses epsilon: if random number < ε → random action (explore), else → pick highest Q-value (exploit).
store_transition(s, a, r, s', done)
Saves one experience to the Replay Buffer. Buffer holds the last 10,000 experiences.
train_step()
Samples 32 random experiences → computes TD error → backpropagates → updates weights via Adam.
maybe_update_target()
Every 500 steps, copies policy_net → target_net. Keeps training stable.
epsilon()
Returns exploration rate. Starts at 1.0 (fully random), linearly decays to 0.05 over 20% of training.
🌐 SumoEnv (sumo_env.py)
reset()
Starts a fresh SUMO simulation. Returns the initial observation (vehicle counts on 4 lanes).
step(action)
Sets the traffic light phase, runs 5 sim-seconds, returns (observation, reward, terminated, truncated, info).
_get_obs()
Calls SUMO to count vehicles on each of the 4 lanes entering B1. Returns array shape (4,).
_set_phase(action)
Translates 0 → "GGggrrrr…" (N-S green) or 1 → "rrrrGGgg…" (E-W green) and sends to TraCI.
close()
Disconnects TraCI and shuts down SUMO process.
🚦 SUMO + TraCI (sumo_utils.py)
traci.start(cmd)
Launches SUMO as a subprocess and opens a TCP socket so Python can control it live.
trafficlight.setRedYellowGreenState()
Instantly applies a phase string (e.g. "GGggrrrr…") to intersection B1.
simulationStep()
Advances simulation by 1 second. Called 5 times per agent step (control_interval = 5).
lane.getLastStepVehicleNumber()
Returns vehicle count on a lane in the last second. Used to build the observation array.
vehicle.getWaitingTime()
How long a vehicle has been stopped. Summed over all vehicles → used as reward signal.

DQN RL
Deep Q-Network. A neural network trained to predict the "value" of each possible action. The agent always picks the action with the highest predicted value.
SumoEnv ENV
A Python class wrapping SUMO as a Gymnasium environment. It gives the agent a clean interface: observe → act → get reward — hiding all SUMO complexity.
SUMO SIMULATOR
"Simulation of Urban MObility". Open-source software that simulates cars on a virtual road network with real physics (speed, acceleration, stopping).
TraCI SUMO
"Traffic Control Interface". The Python API to talk to SUMO in real time — read vehicle positions, set traffic lights, advance the clock.
Observation RL
What the agent "sees" at each step. Here: 4 numbers — vehicle count on each incoming lane at intersection B1.
Action RL
What the agent decides. Two choices: 0 = give green to North-South traffic, 1 = give green to East-West traffic.
Reward RL
Score given after each action. Reward = −(sum of all vehicles' waiting time). Minimizing waiting time = maximizing reward.
Replay Buffer NN
Memory storing up to 10,000 past experiences. Agent randomly samples from it during training so it doesn't forget old lessons.
Policy Net NN
The neural network being actively trained. It decides what action to take. Updated every 4 steps.
Target Net NN
A frozen copy of the policy net used only to compute training targets. Synced with policy net every 500 steps. Prevents unstable feedback loops.
Epsilon (ε) RL
Exploration rate. ε = 1.0 → fully random (agent explores freely). ε = 0.05 → mostly uses neural net (agent exploits knowledge). Decays linearly during training.
Intersection B1 SUMO
The center intersection in the road grid. Has 4 incoming lanes (N, S, E, W). The only traffic light the agent controls.
Episode RL
One full simulation run (360 simulated seconds = 72 agent steps). When it ends, SUMO resets and a new episode starts from scratch.
Adam Optimizer NN
Algorithm that updates the neural network weights. It adapts the learning rate per parameter based on past gradients — faster and more stable than plain gradient descent.
TD Error NN
"Temporal Difference Error". The gap between what the agent predicted and what it should have predicted. The loss function minimizes this gap.
Gamma (γ = 0.99) RL
Discount factor. Future rewards are worth 99% as much as immediate ones. High γ = agent cares about long-term consequences, not just right now.
