From-scratch training loop

flowchart TD
  START([Start]) --> RESET[env.reset]
  RESET --> ACT[agent.act: ε-greedy]
  ACT --> STEP[env.step]
  STEP --> STORE[store_transition]
  STORE --> TRAIN[train_step if schedule allows]
  TRAIN --> TARGET{timestep % target_interval == 0?}
  TARGET -->|yes| COPY[target_net ← policy_net]
  TARGET -->|no| DONECHK
  COPY --> DONECHK{terminated or truncated?}
  DONECHK -->|yes| RESET
  DONECHK -->|no| ACT

One DQN gradient step

flowchart TB
  B[Sample minibatch] --> F1[Forward policy_net: Q_theta]
  B --> F2[Forward target_net: Q_theta-]
  F1 --> PICK[Select Q for taken actions a]
  F2 --> MAX[max over actions on s']
  PICK --> LOSS[MSE vs y = r + gamma * max * 1-done]
  MAX --> LOSS
  LOSS --> BP[Backprop to gradients]
  BP --> CLIP[Global L2 clip]
  CLIP --> ADAM[Adam update theta]

After Adam, the target network is updated on a slower clock (every N environment steps), not every gradient step.

ASCII: TD target (scalar per transition)

    s ──► Q_θ(s,·) ──► pick index a ──► Q_θ(s,a)
                                              │
                                              │  compare
                                              ▼
    r + γ·(1-done)·max_a' Q_θ⁻(s',a')  ════════╪══►  MSE loss
                                              │
    s' ──► Q_θ⁻(s',·) ──► max over a' ────────┘

    θ⁻ ← copy(θ)  every `target_update_interval` steps (hard update, τ=1 in SB3)

ASCII: He init vs Adam (intuition)

  He init (ReLU layer, fan_in = n):
       W ~ Normal(0, σ²),  σ = √(2/n)
       keeps variance stable through ReLU blocks at start

  Adam (per weight):
       m ← β₁·m + (1-β₁)·g        ← momentum of gradients
       v ← β₂·v + (1-β₂)·g²       ← running avg of squared grads
       θ ← θ - α · m̂ / (√v̂ + ε)  ← adaptive step (bias-corrected m̂, v̂)
